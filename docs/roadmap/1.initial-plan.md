# 1. 初始实现路线图

本文件用于汇总 `code-dance` 第一阶段的完成情况与剩余缺口。

## 当前结论

第一阶段主链路已经跑通，项目已不再是“Rust 单语言按周采样原型”，而是一个具备本地注册、异步分析、SQLite 持久化、查询接口和 Web 展示的可运行版本。

当前已落地的核心能力：

- [x] monorepo 基础目录结构
- [x] 共享领域模型与 API 契约层
- [x] Git 仓库探测、历史读取与采样能力
- [x] 模块识别框架
- [x] Rust workspace / crate 模块识别
- [x] Node/Web workspace / package 模块识别
- [x] `weekly` / `daily` / `per-commit` 采样
- [x] 模块级 `loc`、`added`、`deleted`、`churn`
- [x] SQLite 存储层
- [x] 基础分析任务 API 与查询 API
- [x] 分析任务进度持久化与轮询展示
- [x] 第一版 Web 可视化页面
- [ ] 最小 TUI 浏览能力
